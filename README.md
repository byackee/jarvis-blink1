## Description
Ce plugin permet de controler les clef USB Blink1 (https://github.com/todbot/blink1)

Les commandes suivantes sont supportées:
 - allume led
 - eteint led
 - led bleue
 - led verte
 - led blanche
 - led jaune
 - led orange

## Usage
```
You: allume led
Jarvis: led allumée
You: regle led avec la couleur bleue
Jarvis: led bleue
```
